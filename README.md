### Overview
This project visualizes ocean flow and temperature by projecting oceanographic data onto a texture and applying a 2D Line Integral Convolution (LIC) technique to acquire the flow visualization.
### Prerequisites
- **OpenGL**: The project uses OpenGL for rendering shaders.
- **Shader Program**: You need a GLSL-compatible shader pipeline for running the `lic2d.frag`, `lic2d.vert`. In this project, I use glman https://web.engr.oregonstate.edu/~mjb/glman/ 
- **C Compiler** (optional) For running the C script (`script.c`), ensure you have a working C compiler like GCC or Clang.
